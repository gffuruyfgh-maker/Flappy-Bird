# Flappy Bird 风格游戏 Demo - 规格说明

## 1. 项目概述
- **项目名称**: Flappy Bird Demo
- **类型**: 2D 街机游戏
- **平台**: Web (HTML5 Canvas)
- **核心玩法**: 控制小鸟穿越管道障碍物得分

## 2. 视觉规格

### 画布设置
- **尺寸**: 400 x 600 像素
- **背景**: 渐变天空色 (#87CEEB → #E0F6FF)

### 角色 - 小鸟
- **形状**: 椭圆 30x24 像素
- **颜色**: 橙黄色 (#FFD93D) 带橙色边框
- **眼睛**: 白色圆点 + 黑色瞳孔
- **位置 X**: 固定在画布 1/4 位置 (100px)

### 管道
- **宽度**: 60 像素
- **颜色**: 绿色 (#2ECC71) 带深绿色边框
- **间隙**: 150 像素
- **生成间隔**: 1.8 秒

### 地面
- **高度**: 80 像素
- **颜色**: 棕色 (#8B4513) 带纹理线条

## 3. 物理规格

| 参数 | 数值 |
|------|------|
| 重力加速度 | 0.5 px/frame² |
| 跳跃速度 | -8 px/frame |
| 终端速度 | 10 px/frame |
| 管道移动速度 | 3 px/frame |

## 4. 游戏状态

### 状态机
```
start → (空格/点击) → playing → (碰撞) → gameover
                                          ↓
                                    (空格/点击)
                                          ↓
                                        start
```

### 状态说明
- **start**: 显示 "Click or Press Space to Start"
- **playing**: 游戏进行中，实时渲染
- **gameover**: 显示 "Game Over"、当前得分、最高分

## 5. 计分系统
- **得分规则**: 每穿越一个管道间隙 +1 分
- **最高分**: 存储在 localStorage，键名 `flappyBestScore`

## 6. 控制方式
- **空格键**: 跳跃
- **鼠标点击**: 跳跃
- **触摸**: 跳跃

## 7. 验收标准
- [ ] 小鸟受重力影响自然下落
- [ ] 点击/空格能控制小鸟跳跃
- [ ] 管道从右侧生成并向左移动
- [ ] 碰撞管道/地面/天花板后游戏结束
- [ ] 穿越管道间隙时得分 +1
- [ ] 最高分正确保存和显示
- [ ] 游戏结束后可重新开始
